頂点数分ループ{
	Position(ローカル座標)
	Normal(法線)
	Tangent(接空間に変換する行列)
	TextureCoordinates(テクスチャ座標)
	
	色があれば
		color(色 UINT型)
	無ければ
		color = 0xFFFFFFFF
	
	それぞれ配列に用意
}

スキンがあれば
頂点数分ループ{
	４つのボーンの、インデックスとウェイトを配列に用意
	つまり一度に影響をうけられるボーンは４つまで。
}

インデックス分ループ{
	インデックスを配列に用意
}

ループの前には要素数書き込みがある

メッシュ数分ループ{

	メッシュの名前長と名前を書き込む

	マテリアルが無ければデフォルトのものを追加しておく

	マテリアル数分ループ{
		マテリアル名
		アンビエントカラー
		デフューズカラー
		スペキュラーカラー(なければデフォルト設定がある)
		エミッシブカラー
		UVトランスフォーム
		
		これらを書き込む
		
		スペキュラがあれば
			ピクセルシェーダー名にphong.dgslを
		なければ
			ピクセルシェーダー名にlambert.dgslを
		書き込む
		
		テクスチャパスを最低1個、最大8個書き込む。
		反射や凹凸などの特殊なテクスチャのために複数使ったりするんだろう。
	}

	スケルトン、アニメーションが無ければ０、あれば１を書き込む。

	サブメッシュ数分ループ{
		サブメッシュの
		マテリアルインデックス
		インデックスバッファのインデックス
		頂点のインデックス
		スタートインデックス(サブメッシュの頂点インデックスの最初？)
		プリムカウント(サブメッシュのポリゴンの数？)
		を書き込む
	}
	
	インデックス
	
	頂点
	
	スキニング頂点
	
	それぞれ書き込む
}
